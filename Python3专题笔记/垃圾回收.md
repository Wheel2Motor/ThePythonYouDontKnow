垃圾回收
================================================================================

Python中主要使用的内存回收方式是引用计数，这种方式性能好，实现简单。

但是一旦遇到循环引用，那么两个相互引用的对象谁都无法到达计数归零的时候，此时会在特定时机触发GC（Garbage Collection）。或者主动触发GC `gc.collect()` 。

GC是一个比较费性能的特性，在一些大量对象产生与消失、同时性能敏感的内容里，可以临时关闭GC。

```python
import gc
import time

class Dummy1:
    def __init__(self):
        self.dymmy2 = None

class Dummy2:
    def __init__(self):
        self.dymmy1 = None

def perfconsum(cnt):
    for i in range(cnt):
        d1 = Dummy1()
        d2 = Dummy2()
        d1.dummy2 = d2
        d2.dummy1 = d1

def timer(func, *args, **kwargs):
    beg = time.time()
    res = func(*args, **kwargs)
    end = time.time()
    print(end - beg)
    return res

gc.disable()
timer(perfconsum, 10000) # 0.39

gc.enable()
timer(perfconsum, 10000) # 0.80
```
